# We Designerz — документация пивота

Сайт переходит от «студии дизайна» к **сообществу вайбкодеров**. Этот пакет — источник
правды для дальнейшей работы. Sonnet реализует код по `04-tasks-sonnet.md`.

## Решения (зафиксировано)
- Полный пивот в сообщество. Платная услуга — не ядро, остаётся как доказательство
  экспертизы + один hire-CTA.
- Бэкенд: **Supabase** (free). Сайт остаётся статикой на GitHub Pages.
- Чат: **Telegram**.
- Язык: **RU** на запуск, архитектура **EN-ready**.

## Миссия (кратко)
Самое большое сообщество вайбкодеров в СНГ. Не витрина — **клуб**, членством в
котором гордятся; вайбкодинг — новое профессиональное направление. Детали —
`00-vision.md` (миссия, семантика «Вступить») и `10-membership.md` (грейды).

## Минимум до анонса
Задачи T0–T8 + T10 + T11 + T-UX1 + T15 (членство и доверие) + сидинг витрины
и чата. Детали — в `04`, `05` и `09`.
