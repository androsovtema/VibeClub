## Цель
Заменить фейковые захардкоженные карточки на реальные проекты из Supabase (таблица `projects`,
только `status='published'`), и оформить карточку в стиле превью видео на YouTube. Это касается
двух мест: витрина на главной (`index.html`, контейнер `#community-grid`) и полная витрина
`projects.html`.

## A. Данные — `js/projects.js`
- `fetchPublishedProjects({ category, sort, limit })`:
  - Supabase: `from('projects').select('*, author:profiles(display_name, avatar_url), comments(count)')`
    `.eq('status','published')`.
  - `category` (опц.): фильтр по тегу — проект попадает, если его `tags` содержит категорию.
    Категории (единые с чипами на главной): `prod` (Продуктивность), `biz` (Бизнес),
    `game` (Игры), `home` (Личное), `art` (Творчество). `all` — без фильтра.
  - `sort`: `new` → `order('created_at', {ascending:false})`; `top` → `order('upvotes', {ascending:false})`.
  - `limit` (опц.) — для главной (напр. 8).
  - Возвращай нормализованные объекты (title, description, cover_url, project_url, tags, tools,
    is_core, upvotes, commentsCount, author.display_name, author.avatar_url, id).
- `fetchProjectById(id)` — на будущее (T4). Достаточно заготовки с тем же select + `.eq('id', id).single()`.

## B. Карточка в стиле YouTube — `renderProjectCard(project)` в `js/projects.js`
Возвращает DOM-узел (или HTML-строку с ОБЯЗАТЕЛЬНЫМ экранированием, см. «Безопасность»):
- **Обложка 16:9**: `cover_url` картинкой; если нет — градиент-плейсхолдер (как сейчас в
  `main.js`, из палитры токенов). В углу обложки — **бейдж инструмента** (первый из `tools`,
  моно-шрифт), как «длительность» у видео. Если `is_core` — метка `✦` (команда We Designerz).
- **Под обложкой:** строка автора = аватар (кружок: `avatar_url` или инициалы, если нет) +
  имя автора. Заголовок проекта — до 2 строк (`line-clamp: 2`).
- **Мета-строка:** `♥ {upvotes}` · `💬 {commentsCount}` · тег(и). Компактно.
- Вся карточка — ссылка/кликабельна на `project.html?id={id}` (страница проекта — T4;
  пока просто ведёт туда, 404-контент допустим).
- Стиль — на токенах `css/tokens.css`, тёмная тема. Заменяет прежнюю `.community-card`.
  Не «типовая карточка» — привычная видео-плитка.

## D. Полная витрина `projects.html`
`projects.html` сейчас — СТАРАЯ портфолио-страница дизайн-студии (хардкод `.project-card`,

- Фильтры — те же категории сообщества (`all/prod/biz/game/home/art`) + сортировка

## Безопасность (обязательно)
- Любой текст из БД (название, описание, имя автора, теги, инструменты) при вставке в DOM
  ЭКРАНИРУЕТСЯ. Не суй сырой `innerHTML` с данными пользователя. Вынеси `escapeHtml` из
  `js/app.js` в общий `js/util.js` и переиспользуй (или строй узлы через `textContent`/
  `createElement`). Это защита от хранимого XSS.
- `service_role` не использовать. Только `anon` из `config.js`. Только ESM + ванильный JS.
